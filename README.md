# Credit-Wise-Loan-System
Loan Approval System - Minor Project
Initially done on Jupyter lab, Credit wise loan system is a loan approval system which is based on supervised learning, it interprets the mertrics/features to predict if your loan can be approved or not (yes/no). 
The project was more of anlayisising and cleaning the dataset, performing EDA, feature engineering and model selection with evalutaion (precision,Recall,F1-score) to see which model returns the best results. 

Models used - KNN, Logistic Regression and Naive Bayes.
